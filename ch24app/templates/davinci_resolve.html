How to Export Media from DaVinci Resolve

Here's a step-by-step guide to export your video in DaVinci Resolve to meet the specified standards:

1. Set Up Your Project Settings

  • Go to File > Project Settings.
  • Under Master Settings, set the Timeline Resolution to 1280x720 for 720p or 1920x1080 for 1080p.
  • Set the Timeline Frame Rate to 29.97 fps.
  • Ensure the Pixel Aspect Ratio matches 16:9 or 4:3.

2. Go to the Deliver Page

  • Once your timeline is ready for export, navigate to the Deliver page at the bottom of the screen.

3. Export Settings

  • Format: Choose MP4 for .mp4 or .m4v, or QuickTime for .mov.
  • Codec: Set to H.264.
  • Resolution: Select 1280x720 for 720p or 1920x1080 for 1080p, based on your requirement.
  • Frame Rate: Confirm it is set to 29.97 fps.

4. Bitrate Settings

  • Scroll down to the Quality section.
  • Choose Restrict to and set the bitrate:
    • 720p: Set between 5,000–7,500 kbps (5–7.5 Mbps).
    • 1080p: Set between 8,000–12,000 kbps (8–12 Mbps).

5. Audio Settings

  • Go to the Audio tab within the Deliver page.
  • Audio Codec: Set to AAC.
  • Channels: Select Stereo.
  • Bitrate: Choose 192 kbps or 256 kbps.
  • Sample Rate: Set to 48,000 Hz.

6. File Naming

  • Enter a file name using only letters, numbers, dashes, and underscores.
  • Avoid quotes ("), ampersands (&), slashes (/), and special characters.

7. Add to Render Queue and Export

  • Click Add to Render Queue on the right side of the Deliver page.
  • Then, select Render All to start exporting your file.

By following these instructions, you'll have a video exported to meet the required format, codec, resolution, bitrate, frame rate, and audio standards.
